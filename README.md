# Vanguard Command Hub
Cinematic sci-fi warship interface. Built with pure HTML/CSS/JS.

## Features
* **Live Canvas Animation**: Stars, ships, lasers, and debris.
* **Monochrome Style**: Clean, modern dark mode.
* **Dashboards**: Threat starcharts, roster filters, and terminal logs.

## Setup
Double-click `index.html` to run offline.
